# Philosophy

Agent Connections Protocol treats relationship memory as infrastructure.

Agents should not make trust decisions from global reputation alone. They should combine public claims, third-party attestations, prior experience, task scope, and stakes.

Markdown is the source of truth because it is easy for humans and agents to read, edit, review, diff, and commit.

The protocol starts small:

- no server
- no database
- no required registry
- no required blockchain
- no hidden state

Databases, graphs, and registries can be derived later. The durable record begins as readable files.
